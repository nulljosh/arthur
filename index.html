What It Knows

Q: What is 7*8?       -> 56
Q: What is your name? -> core
Q: Who made you?      -> Josh made me
Q: print hello world  -> print "Hello, World!";
Q: write a function   -> fn add(a, b) { return a + b; }

Stack

  • Tokenizer: Character-level (102 chars)
  • Architecture: Decoder-only transformer (attention, FFN, layer norm, residuals)
  • Training: AdamW, cosine LR, persistent daemon with RAM guard
  • Corpus: 185 KB mixed Q&A (math, identity, jot code, facts)
  • Web UI: Flask chat interface, Apple Liquid Glass design

Run It

git clone https://github.com/nulljosh/core
cd core
python3.13 -m venv venv && source venv/bin/activate
